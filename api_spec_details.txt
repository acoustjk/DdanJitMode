[공공데이터포털] 교통안전 신호등 실시간 정보 Open API 명세 (Base: B551982/rti)
========================================================================

본 문서는 행정안전부 한국지역정보개발원(KLID)에서 제공하는 실시간 신호등의 API 정보 데이터 범위를 명세합니다.

------------------------------------------------------------------------
1. 교차로 목록 조회 API (getItstList)
------------------------------------------------------------------------
지자체 내 등록된 교차로의 위치와 속성 데이터를 얻기 위한 API입니다.

* 제공 필드 범위:
  - itstId (String)     : 교차로 고유 식별 ID
  - itstNm (String)     : 교차로 한글 이름 (예: "강남역 사거리")
  - la (Double)         : 교차로 위치의 GPS 위도 (Latitude)
  - lo (Double)         : 교차로 위치의 GPS 경도 (Longitude)
  - ltcCode (String)    : 관할 지자체 행정 코드
  - limitSpeed (Integer): 도로 제한 속도 (km/h)
  - laneWidth (Double)  : 평균 차로 폭 (m)

------------------------------------------------------------------------
2. 신호등 실시간 잔여시간 정보 조회 API (getSgList)
------------------------------------------------------------------------
특정 교차로의 실시간 직진, 좌회전, 보행 신호등 점등 시간 및 잔여초 정보입니다.

* 제공 필드 범위:
  - itstId (String)     : 소속된 교차로의 식별 ID (getItstList의 ID와 매핑)
  - sgId (String)       : 개별 신호등 고유 식별 ID
  - color (String)      : 현재 신호등 색상 위상 (RED: 적색, GREEN: 녹색, YELLOW: 황색)
  - tr (Integer)        : 현재 신호가 유지되는 남은 잔여 시간 (초 단위)
  - dir (String)        : 신호 방향 코드 (1: 북, 2: 동, 3: 남, 4: 서, 5: 북동, 6: 남동, 7: 남서, 8: 북서)
  - sgType (String)     : 신호 속성 유형 (Straight: 직진, Left: 좌회전, Pedestrian: 보행, Uturn: 유턴)

------------------------------------------------------------------------
3. 서비스 유의사항
------------------------------------------------------------------------
* 지자체의 신호 제어기 상태가 실시간으로 통신망을 통해 올라오므로, 로컬 환경 및 서버 트래픽 상황에 따라 실신호와 1~2초 내외의 시간 오차가 발생할 수 있습니다.
* "tr(잔여시간)" 데이터는 신호가 바뀌기 직전까지 지속적으로 감산되며, 0초가 도달하는 즉시 "color" 필드가 교체 갱신됩니다.
